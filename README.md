## 🚀 How to Use

1. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the tool using:
   ```bash
   python DiscordSpammer_CLI.py
   ```
   *or*  
   Execute the pre-configured batch file:
   ```bash
   run.bat
   ```
